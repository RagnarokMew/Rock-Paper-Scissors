# Rock-Paper-Scissors
A website, where you can play RPS (Rock Paper Scissors) all by yourself. Part of the Odin Project curriculum.
### Goal
* Basic RPS Game
* Use DOM Manipulation
* Include Basic Animations (CSS and/or JS)
* Prove understanding of basic JS
